# JSON Format Converter

A Python utility that converts IoT device telemetry payloads between two JSON formats into a normalised output structure.

## How to run

```
python main.py
```

The script runs a built-in test suite and prints `All tests passed` if everything is working.

## Project structure

- `main.py` — two converter functions (`convertFromFormat1`, `convertFromFormat2`) and a self-contained test runner.

## Workflow

The **Start application** workflow runs `python main.py` (console output).
